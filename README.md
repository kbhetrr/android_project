
# android_project
2025-1학기 소프트웨어 설계 및 실험 과목 텀프로젝트

## 📝 Git 커밋 타입 가이드

| 타입      | 설명 |
|-----------|------|
| `feat`    | ✨ 새로운 기능에 대한 커밋 |
| `fix`     | 🐛 버그 수정에 대한 커밋 |
| `build`   | 🔧 빌드 관련 파일 수정, 모듈 설치 또는 삭제 |
| `chore`   | 🔨 그 외 자잘한 수정 (빌드, 패키지 매니저 설정 등) |
| `ci`      | 🤖 CI 설정 관련 수정 |
| `docs`    | 📝 문서 수정 (README 등) |
| `style`   | 💄 코드 포맷, 세미콜론 누락, 들여쓰기 등 스타일 수정 |
| `refactor`| ♻️  코드 리팩토링 (기능 변화 없음) |
| `test`    | ✅ 테스트 코드 추가 또는 수정 |
| `perf`    | ⚡ 성능 개선 관련 코드 수정 |
